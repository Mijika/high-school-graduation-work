컴퓨터 프로그램 정의
메인 화면 = 컴퓨터 로봇 제어 프로그램은 총 두개의 스크린을 사용한다.
		 그중 메인 화면은 로봇에서 오는 영상 스트레밍을 스크린 전체에 송출한다.
		 또한 키보드와 마우스를 이용한 버튼 클린을 통한 로봇의 모션을 제어한다.
지도 화면 = 컴퓨터 로봇 제어 프로그램은 총 두개의 스크린을 사용한다.
		 그중 지도 화면은 로봇에서 오는 GPS값을 구글 맵에 매칭하여 현재 위치를 
		 위성 지도에 나타내고 그걸 스크린 전체에 송출한다. 또한 화면 우측 상단에 현재 GPS값과 고도 값을 송출한다.
백그라운드 = 컴퓨터가 로봇으로 부터오는 정보들을 수신하는 곳이다.
		 또한 로봇에 명령를 내리면 백그라운드를 통해 명령이 전해진다.
		 구글 맵의 지도 화면 등을 가져오는 것 역시 요기에서 수행한다.
	메인 화면
		풀 스크린  카메라 스트레밍
		키 입력을 통한 로봇의 이동과 높이 변형	
		특정 키와 버튼을 통한 LED 전등	
		특정 키와 버튼을 통한 부저나 특정 소리 재생
		
	지도 화면
		현재 로봇의 GPS 위치를 표시 (경도 위도 높이)
			-(GPS가 안 통할 경우 고도 센서를 통한 고도 확인)
		특정 버튼을 통한 그 동안 지나온 경로 표시

	백그라운드 
		ESP-cam과 실시간 스트레밍 연결
		ESP-mobil과 모션및 행동 연결
 
로봇 기능 정의
	ESP-CAM = 로봇에 달린 2개의 ESP 중 하나로 실시간으로 영상 송출을 담당 한다.
	ESP-MOBIL = 로봇에 달린 2개의 ESP 중 하나로 로봇의 모션및 센싱값의 제어와 송출을 담당한다.

	ESP-CAM
		지속적으로 연결된 와이파이 내에서 웹 서버를 실행하면서 스트레밍 영상을 송출한다.
	ESP-MOBIL
		이동 모터의 제어
		바퀴의 체고 조정 모터?서보모터를 제어
		전방-상방 LED 등 제어
		현제 GPS 값 실시간을 전송
		현제 고도계 센서값 실시간으로 전송
